What can you do with Kafka Topology Builder
*******************************

The main purpose of the Kafka Topology Builder is to help automating boring administrative tasks that each team dealing with Kafka have to do.
This tasks usually are around Topics, Access Control, but as well on handling Schemas and more.

In this chapter we will introduce the different things one can configure in each topology file(s) and how to take advantage of the tool the best.
